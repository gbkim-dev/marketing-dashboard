# data/ — 일별 마케팅 데이터 폴더

## 매일 작업 흐름

1. 오늘 날짜 폴더 만들기: `data/YYYY-MM-DD/`
2. 채널 CSV → 그 폴더에 **`channel.csv`** 이름으로 저장
3. 앱스플라이어 CSV → 같은 폴더에 **`appsflyer.csv`** 이름으로 저장
4. 끝. 대시보드 새로고침하면 자동 반영 (60초 캐시)

## 표준 폴더 구조

```
data/
├── 2025-01-01/
│   ├── channel.csv         ← 매체 보고 데이터
│   └── appsflyer.csv       ← AppsFlyer 어트리뷰션 데이터
├── 2025-01-02/
│   ├── channel.csv
│   └── appsflyer.csv
├── _archive/               ← 오래된 데이터 (자동 무시됨)
│   └── 2024/
└── README.md
```

## 파일명 규칙 (둘 다 지원)

### ✅ 권장 (날짜 폴더)
- `data/2025-01-01/channel.csv`
- `data/2025-01-01/appsflyer.csv`

### 🔁 호환 (Flat — 워크샵 샘플 호환)
- `data/2025-01-01_channel.csv`
- `data/2025-01-01_appsflyer.csv`

## 제외 규칙

- `_`로 시작하는 폴더(`_archive/`, `_old/` 등)는 스캔에서 제외됨
- 임시·삭제 파일 보관용으로 사용

## 컬럼 표준

### channel.csv (매체 보고)
`일 · 채널 · 채널분류 · 캠페인 · 캠페인목적 · 그룹 · 소재 · 노출 · 클릭 · 비용 · 회원가입 · 구매 · 구매매출`

### appsflyer.csv (어트리뷰션)
`일 · 미디어소스 · 캠페인 · 그룹 · 소재 · 클릭 · 회원가입 · 구매 · 구매매출`

## 조인 키 (4개)

`일 · 캠페인 · 그룹 · 소재` — 두 CSV가 이 4개로 매칭됨
